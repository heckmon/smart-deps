### Supported frameworks
  - [npm](https://www.npmjs.com/)
  - [deno](https://deno.com/blog/your-new-js-package-manager)
  - [bun](https://bun.com/package-manager)
  - [pub](https://pub.dev/)
  - [pypi](https://pypi.org/)
  - [pipenv](https://pypi.org/project/pipenv/)
  - [poetry](https://python-poetry.org/)
  - [uv](https://docs.astral.sh/uv/)
  - [cargo](https://crates.io/)
  - [maven](https://mvnrepository.com/)
  - [gradle](https://plugins.gradle.org/search?term=package+manager)
  - [nuget](https://www.nuget.org/)
  - [composer](https://getcomposer.org/)
  - [gem](https://rubygems.org/)
  - [go_modules](https://pkg.go.dev/)
  - [swift_pm](https://docs.swift.org/swiftpm/documentation/packagemanagerdocs/)
  - [conan](https://conan.io/)
  - [vcpkg](https://vcpkg.io/en/)
  - [luarocks](https://luarocks.org/)
  - [hex](https://hex.pm/)
